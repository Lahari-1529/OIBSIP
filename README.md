OIBSIP - Python Projects

This repository contains the Python projects completed as part of the Oasis Infobyte Internship.

Projects Included:

1. BMI Calculator (Task 1)
* File Name: Lahari_Task1.py
* Description: A simple command-line application that calculates Body Mass Index (BMI) based on user input for weight and height, categorizing the health status.

2. Password Generator (Task 2)
* File Name: Lahari_Task2.py
* Description: A tool that generates strong, random, and secure passwords based on user preferences (length, inclusion of numbers, symbols, etc.).

3. Chat Application (Task 3)
* File Names: Lahari_Task3_client.py and Lahari_Task3_server.py
* Description: A simple client-server chat application built using Python sockets, allowing real-time communication.

Requirements and Installation:

To run these projects locally, you need Python installed on your system.

1. Clone this repository:
git clone https://github.com/Lahari-1529/OIBSIP.git

2. Navigate to the project directory:
cd OIBSIP

3. Run any task (Example - Task 1):
python Lahari_Task1.py
